# Система мониторинга веб-приложения

Система мониторинга для веб-приложения на Go с автоматическим перезапуском при сбое.


## Компоненты системы

### 1. Веб-приложение (Go)

Простой HTTP сервер с 2 endpoint:
- `/` - возвращает "Hello World!"
- `/health` - для проверки здоровья приложения
По умолчанию используется порт 8080

### 2. Скрипт мониторинга (Bash)

Функционал:
- Проверка доступности приложения через HTTP запрос к `/health`
- Логирование всех проверок и действий
- Автоматический перезапуск при недоступности
- Компиляция приложения при отсутствии бинарного файла

### 3. Systemd сервисы

- **webapp.service** - управление жизненным циклом приложения
- **webapp-monitor.timer** - периодический запуск проверки (каждые 30 секунд)
- **webapp-monitor.service** - однократный запуск скрипта мониторинга

## Требования

- Linux с systemd
- Go
- curl
- Права root для установки

## Установка & Запуск

```bash
# Клонируйте репозиторий
git clone 
cd webapp-monitoring

# Запустите установку
sudo ./scripts/install.sh
```
